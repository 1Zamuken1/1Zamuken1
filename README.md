## Proyectos Destacados

### GastuApp - Gestor de Finanzas con IA

**Sistema de gestión financiera con integración de inteligencia artificial**

- Backend en Spring Boot con patrón MVC
- Integración de IA para asesoría financiera automatizada (GROQ API)
- API REST con endpoints para transacciones, cálculo de balance y generación de reportes
- Interfaz web con Thymeleaf

**Stack**: Java 21, Spring Boot, Spring Data JPA, MySQL, Thymeleaf, REST APIs

[Ver repositorio](https://github.com/1Zamuken1/GastuApp)

### Salsamentaria - E-commerce Full-Stack

**Proyecto académico: tienda online con sistema de autenticación y gestión de roles**

- Backend: Spring Boot con Spring Security y JWT para autenticación
- Frontend: Angular con Bootstrap, dashboard administrativo
- Sistema de roles y permisos (Admin, Usuario)
- CRUD completo de productos, pedidos y usuarios
- Nota: Proyecto de demostración con simulación de pasarela de pago (no producción)

**Stack**: Java 21, Spring Boot, Spring Security, Angular, Bootstrap, JWT, MySQL

[Ver repositorio](https://github.com/1Zamuken1/taller_calidad_software_Salsamentaria)
